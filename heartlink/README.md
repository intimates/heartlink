# Heartlink

## Libraries

### Framework

- Vue.js

### CSS

- Twitter Bootstrap
- Sass

## Development Setup

### Install dependencies

1. `npm install -g browserify beefy`
2. `npm install`
3. `bower install` (I want to remove bower in the future...)

### Start writing code

1. watch js and build automatically: `npm run dev`
2. start server: `beefy src/main.js`
